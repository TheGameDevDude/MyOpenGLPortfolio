# MyOpenGLPortfolio
![](ScreenShots/screenshot_normal.GIF)

### Features

+ OpenGL 3.0+
+ Diffuse lighting
+ Specular lighting
+ Realtime reflections
+ Normal mapping
+ Parallax mapping 
+ Batch rendering 
+ Realtime reflections

# Running the executable
+ Download latest executable from the releases page.
+ The executable only works on windows.
